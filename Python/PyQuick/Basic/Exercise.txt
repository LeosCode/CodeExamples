Basic Python Exercises

There are 3 exercises that go with the first sections of Google's Python class. They are located in the "basic" directory within the google-python-exercises directory. 

string1.py -- complete the string functions in string1.py, based on the material in the Python Strings section 

string2.p -- Additional string exercises 

list1.py -- complete the list functions in list1.py, based on the material in the Python Lists and Python Sorting sections 

list2.py -- Additional list exercises

wordcount.py -- this larger, summary exercise in wordcount.py combines all the basic Python material in the above sections plus Python Dicts and Files 

mimic.py -- Additional combination exercise
